# Learning Roadmap Workflow

Tujuan: membuat roadmap belajar terkontrol, bukan sekadar list materi.

Workflow:
1. Buat draft roadmap
2. Simpan status draft
3. Request approval jika akan aktivasi dan bulk task
4. Setelah approval, task/reminder bisa dibuat oleh eksekutor lanjutan

Commands:
• `/roadmaps`
• `/roadmap`
• `/study-today`
• `/study-review`

Tools: `learning_create_roadmap`, `learning_list_roadmaps`, `learning_generate_today_plan`.

Batasan: aktivasi penuh pasca-approval belum mengeksekusi bulk task otomatis.
