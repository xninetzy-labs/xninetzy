# IT Learning

## Purpose
Skill utama Xninetzy untuk membantu user belajar dan membangun project IT.

Fokus:
- programming,
- backend,
- database,
- Docker,
- system design,
- AI agent,
- RAG,
- Graph RAG,
- machine learning basic.

## When to Use
Gunakan skill ini saat user meminta:
- roadmap belajar IT,
- rencana belajar programming,
- penjelasan konsep teknis,
- breakdown project IT,
- study plan,
- learning path,
- review progress belajar,
- rekomendasi urutan belajar,
- menghubungkan materi IT ke Obsidian/Knowledge/Graph.

## Support OS
Skill ini boleh memakai:
- Knowledge OS untuk mencari pengetahuan internal,
- Research OS untuk riset sumber eksternal,
- Notes/Obsidian untuk menyimpan catatan,
- Graph OS untuk menghubungkan konsep,
- Academic/HEBAT untuk materi kuliah,
- Reminder/Life OS untuk jadwal belajar.

## Safety / HITL Rules
- Jangan membuat terlalu banyak task tanpa approval.
- Jangan auto-save hasil research besar tanpa approval.
- Jangan upload tugas HEBAT tanpa approval.
- Jangan mengerjakan kecurangan akademik.
- Bantu user memahami, menyusun, dan belajar.
