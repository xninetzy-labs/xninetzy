# Deep Research Workflow

Tujuan: riset mendalam WhatsApp-first dengan permission, session, substeps, sumber terpilih, dan brief.

Workflow:
1. Permission check
2. Create research session
3. Notify admin started
4. Plan dan subplan
5. Web/YouTube/academic search action
6. Source ranking
7. Generate brief
8. Mark done
9. Notify admin done

Output berisi tujuan, sub-plan, temuan utama, sumber, YouTube learning path, Graph RAG draft, dan next action.

Batasan: tidak auto-save ke Obsidian/Knowledge/Graph tanpa approval.
