# HITL Admin Approval

Tujuan: aksi berdampak besar tidak dieksekusi tanpa persetujuan admin.

Commands:
• `/approvals`
• `/approve <id>`
• `/reject <id>`

Wajib approval:
• save research to Obsidian/Knowledge
• create roadmap from research
• bulk task/reminder creation
• Graph RAG write dari research besar
• HEBAT upload assignment

Batasan: eksekutor aksi setelah approval masih MVP status update, belum dispatch payload otomatis.
