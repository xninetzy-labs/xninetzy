# IT Learning OS

Domain aktif pertama Xninetzy.

Tujuan:
- membantu user belajar programming, backend, database, Docker, system design, AI agent, RAG, Graph RAG, dan ML basic.
- membuat roadmap belajar IT.
- membuat study plan harian/mingguan.
- menyimpan catatan belajar ke Obsidian.
- menghubungkan catatan ke Knowledge OS dan Graph OS.
- membantu review progress belajar.

Domain lain seperti Biology, IT+Biology, dan Neuroscience belum aktif penuh. Folder `future/` hanya placeholder.

## Isi domain

- `skill_tree.py` — peta skill IT yang dipakai untuk klasifikasi & roadmap.
- `roadmap_models.py`, `roadmap_planner.py`, `roadmap_store.py` — model & engine roadmap.
- `roadmap_tools.py` — tools roadmap yang terdaftar di tools registry.
- `progress_tracker.py` — tracking progress belajar.
- `study_session.py` — sesi belajar.
- `templates.py` — template pembelajaran.

OS pendukung (Knowledge, Research, Graph, Notes/Obsidian, Academic/HEBAT, Life,
Reminder) berada di `app/xninetzy/os/*` dan `app/xninetzy/interfaces/*`.
