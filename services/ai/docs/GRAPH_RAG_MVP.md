# Graph RAG MVP

Tujuan: SQLite node-edge graph untuk menghubungkan topik, sumber, note, task, goal, roadmap, HEBAT, dan research brief.

Tables:
• `graph_nodes`
• `graph_edges`

Tools:
• `graph_add_node`
• `graph_add_edge`
• `graph_search`
• `graph_get_context`
• `graph_explain_topic_map`

Batasan: belum Neo4j dan belum hybrid semantic graph retrieval penuh.
